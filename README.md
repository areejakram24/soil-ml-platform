# Soil Analytics & Crop Prediction Platform
A production-grade, full-stack MERN application integrated with machine learning layers to analyze soil telemetry data and predict optimized crop yield recommendations. The platform features a responsive UI built with Tailwind CSS v4, secure user sessions via JWT, and automated cloud-native CI/CD pipelines.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS v4, Vite, Context API
- **Backend:** Node.js, Express.js, REST APIs, JWT (JSON Web Tokens)
- **Database:** MongoDB Atlas (Cloud Managed Cluster)
- **DevOps & Hosting:** Git, Vercel (Frontend), Render (Backend)

---

## Features

- **7-Metric Telemetry Processing:** Analyzes complex soil health vectors (N, P, K, pH, moisture, and others) to determine ideal cultivation environments.
- **Secure Gateway:** Automated session management utilizing JWT tokens and type-safe React Context hooks to protect private user routing.
- **Automated CI/CD:** Native Git-triggered compilation pipelines that build, test, and deploy code updates to production in under 60 seconds.
